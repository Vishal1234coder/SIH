# SIH - Smart Healthcare Platform

A prototype healthcare management system built for Smart India Hackathon. This is an earlier version of the platform with role-based dashboards for patients, doctors, and administrators.

## Features

- **Role-Based Login**: Super Admin, Hospital Admin, Doctor, Patient
- **Patient Dashboard**: Medicine tracking, compliance monitoring
- **Doctor Dashboard**: Patient management, prescription creation
- **Admin Panel**: Hospital oversight and system analytics
- **AI Assistant Chat**: Basic chatbot integration
- **Dark/Light Theme**: Theme toggle support

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn/ui
- **Build Tool**: Vite
- **Charts**: Recharts

## Getting Started

\`\`\`bash
cd client
npm install
npm run dev
\`\`\`

## Login Credentials (Demo)

Select any role from the login screen and use any email/password combination.
